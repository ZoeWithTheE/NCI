NCI is a client-side Minecraft mod that hooks into the Nexo plugin to provide a seamless way to get the custom items and blocks provided by Nexo into the vanilla creative inventory instead of having to navigate the `/nexo inventory` menu. NCI automatically reflects any changes made to Nexo's inventory, like for instance after running `/nexo reload` to apply changes.

The road map for this companion mod currently includes:

Implementation with JEI/REI/EMI

Vanilla crafting book integration

And more, we are always accepting community feedback!


> Q: Why does NCI exist?

> A: NCI is primarily a tool for creative mode users with access to Nexo's inventory who would like easier access to the items it provides.


> Q: Will NCI stay up to date with Nexo and Minecraft?

> A: Yes, NCI will be maintained for as long as possible.


> Q: How can I support the developer and make suggestions about features?

> A: NCI has a Patreon page, which includes memberships that give access to early access builds that include bug fixes and new features.
